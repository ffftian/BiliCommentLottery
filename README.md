# BiliCommentLottery

> Bilibili评论区抽奖工具

## 功能

- [x] 支持视频稿件AV号/BV号
- [x] 支持专栏稿件CV号
- [x] 支持普通动态号
- [x] 根据开始时间、截止时间筛选评论列表
- [x] 根据评论内容筛选评论列表
- [x] UID重复的评论可只计一次
- [x] 可复制评论网址方便访问
- [ ] 楼中楼评论（可能导致对Bilibili服务器大量访问，舍弃）
